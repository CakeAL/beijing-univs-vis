# Beijing Universities VIS 北京各高校视觉识别系统集合

## 简介

本仓库收集了北京市各个大学的视觉识别系统的下载页（包含校徽，校名等）。

如果需要补充或修改信息，欢迎 fork 本仓库编辑 `README.md` 后并提交 PR，或者提交 issue 。
